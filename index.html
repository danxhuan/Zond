Описание проекта

В качестве результата проекта выступит приложение, которое будет предоставлять пользователям удобный интерфейс для просмотра карт, взаимодействия с ними и получения интересующей информации. Работа будет построена по следующему принципу: обученная модель вначале анализирует предоставленные данные, формирует итоговую карту рельефа, уклонов, водотоков, после этого WEB-приложение реализовывает взаимодействие с пользователем, помогая ему в понятном и удобном режиме пользоваться результатами работы нейросети.

Работа идёт в стиле конвертора изображений.
